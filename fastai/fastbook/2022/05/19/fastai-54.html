ULMfit architecture

The architecture, which first using fine turning got great results in NLP. Consider finetuning, as tweaking functions in such a way when if you are already some values of a, b lever are good and optimal for a particular function. Then tweaking value of c is easier right?

ULMfit archtecture consits of three steps:

  • Training a language models with general dataset like wikipedia. So it get so good in predicting these words. Now in transformers, one big differnce is we use masking instead of predicting next word
  • IMDB lnagage build a language model, build on top of LM for wikipedia
  • In three step, is where model classifier comes and based on this label sentences as postive, negative etc.
